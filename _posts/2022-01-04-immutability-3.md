## 불변함을 적용할 수 있는 대상
- name : 값의 이름
- value : 값 자체

그렇다면 이름을 어떻게 불변하게 할 것인가?

## 값을 불변하게 유지하는 방법

자바스크립트에서 변수가 어떤 값을 가리킬 때, 어떤 값으로 가리키는지 구체적인 동작 방법을 알아야 불변성을 이해할 수 있다.

### 자바스크립트의 데이터 타입

| **Primitive** | **Object** |
| -- | -- |
| number | Object |
| string | Array |
| Boolean | Function |
| Null | -- |
| undefined | -- |
| Symbol | -- |
